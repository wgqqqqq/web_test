# VCoder

VCoder 是面向 Agentic 编程模式全新打造AI协同环境，具备三个核心特性：**AI自主闭环**、**全程可观测**、**工程同构**。

## 三大核心特性

### 1. Agentic（智能体驱动）

描述目标，AI 自主规划并完成：

```
你: "实现用户认证，支持 JWT 和 OAuth2"

VCoder:
├─ 分析项目结构和技术栈
├─ 设计认证架构
├─ 生成代码文件
├─ 编写测试用例
└─ 更新相关配置
```

不是一问一答，而是端到端交付。

### 2. 全程可观测

每一步操作都透明可控：

| 能力 | 说明 |
|------|------|
| 执行轨迹 | 实时查看每一步操作 |
| 决策依据 | 理解 AI 为什么这样做 |
| 影响分析 | 预览变更波及范围 |
| 随时回滚 | 任意节点可撤回 |

### 3. 工程同构

自动分析工程特征，AI 生成的代码天然适配项目：

- 识别技术栈、目录结构、代码风格、设计模式
- 提取规则约束（ESLint、Prettier、AI 规则文件）
- 无需手动配置，生成即可用
